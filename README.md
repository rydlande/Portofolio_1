## Built With

    HTML/CSS, in Visual Studio Code.

## Getting Started

### Installing

Clone the repo:

    gh repo clone rydlande/Portofolio_1

Install the dependencies:

    npm install

### Running

To run the app, run the following commands:

    npm run start
